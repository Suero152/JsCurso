Utilize npm init e faça as configurações que serão aplicadas para o package.json
Agora iremos instalar uma biblioteca e iremos utilizar.

---
Console:
npm install moment --save
---
Será criado uma pasta chamada node modules, com o pacote desejado.
Iremos importar o nosso pacote no index.html e aplicamos no index.js para que seja feita a utilização dentro do HTML.

Git é um sistema de controle sobre versão de aplicativos.
Com ele podemos desenvolver projetos colaborativos com muita mais facilidade.

___
No que o GIT pode te ajudar?
    - Registra todas alterações de arquivo.
    - Permite recuperar versões específicas.
    - Resolve perda de arquivos imporantes e resgata arquivos de versões anteriores.

___
Fluxo de trabalho do GIT.
    - Workdir
        Diretório de aruqivos do projeto no computador.
    - Index
        Área temporária de arquivos do projeto.
    - HEAD
        Aponta para o último commit ( versão salva ) do projetos

O que é o Backend?
------------------
Antes de explicar o que é backend vamos explicar o que é frontend.
O frontend cuida da parte de exibição dos dados para o usuário,
já o Backend cuida desses dados em si, para isso, precisamos de uma estrutura que seja capaz de:
+ Receber e fornecer estes dados
+ Validar
+ Armanzenar
+ Assegurar a consistência.
